Code repo: https://github.com/AcharaChisom/monsters-battle

The game involves a wizard and a group of monsters. They have a fixed amount of lives at the beginning of the game. You click on the ATTACK button to generate random values that represent the power of their attacks. Whomever gets to zero first is the Loser.

The languages used are html, css and javascript.

This project introduced me to a lot of concepts in javascript (classes, objects, high-level-functions, exports and imports and many more). But most of all it forced me to practice my problem solving skills i.e knowing where and when to use and call functions and data required to make the program function as it should.
